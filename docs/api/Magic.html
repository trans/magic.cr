    module Magic

    Overview

    High-level Crystal wrapper for libmagic.

    Use this module for normal Crystal code. It opens libmagic handles, loads the magic database, converts C strings into Crystal strings, raises Magic::Error on failure, and closes handles automatically when using the block APIs.

    Use LibMagic directly when you need exact access to the underlying C API.
